Dijkstra算法

是解决单源最短路径问题。给定G（V,E）和起点s，求从起点s到其他顶点的最短距离。

  • 直接在dijkstra算法里面写

Bellman-Ford算法

他可以解决存在负权边的问题。通常使用邻接表。

SPFA算法

SPFA算法是在Bellman-Ford算法的基础上拓展的。利用了只有d[u]改变了，和u所连接的结点v才会改变。

Floyd算法

Floyd算法是全源最短路径算法。它基于一个这样的思想：如果k作为中介点，使得

d [ i ] [ k ]+d [ k ] [ j ]<d [ i ] [ j ]，那么就更新d [ i ] [ j ]
